How I setup vim on Windows 10. I use gvim on windows because the terminals
behaves oddly and gvim is just as fast.
-------------------------------------------------------------------------------

0. Sharp key remap caps lock to escape
	getting sharp key
	load the file to remap the key

1. Get vim -> download instructions and link whatever its done

2. Setup vimplug with powershell
	iwr -useb https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim |`
	    ni $HOME/vimfiles/autoload/plug.vim -Force`

3. Get the _vimrcfiles from github put the in your %USERPROFILE% directory
	git clone 
	cp
