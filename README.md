Configurations files
====================

Configuration files that are actively maintained:

- [emacs](./emacs/settings.org)
- [bash](./bash/.bashrc)
- [tmux](./tmux/.tmux.conf)
- [xterm](./xterm/.Xresources)
- [vim](./vim/.vimrc)

Modify and run `bash manager.sh` to update the configuration on your system.
